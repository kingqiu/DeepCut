# 局域网访问配置

## 配置说明

开发服务器已配置为支持局域网访问，允许同一网络内的其他设备访问。

## 访问方式

### 本机访问
```
http://localhost:3001
```

### 局域网内其他设备访问
```
http://192.168.28.249:3001
```

**注意**: IP 地址 `192.168.28.249` 是当前本机的局域网 IP，可能会变化。

## 如何获取当前 IP 地址

在 macOS/Linux 上运行：
```bash
ifconfig | grep "inet " | grep -v 127.0.0.1 | awk '{print $2}'
```

或者在系统设置中查看网络信息。

## 配置详情

`package.json` 中的 dev 脚本：
```json
"dev": "next dev -H 0.0.0.0 -p 3001"
```

参数说明：
- `-H 0.0.0.0`: 监听所有网络接口，允许外部访问
- `-p 3001`: 指定端口号为 3001

## 防火墙设置

如果无法访问，请检查：
1. macOS 防火墙是否允许 Node.js 接收传入连接
2. 路由器是否有访问限制
3. 确保两台设备在同一局域网内

## 安全提示

⚠️ **仅在开发环境使用**
- 不要在公网暴露开发服务器
- 生产环境请使用 `npm run build && npm start`
